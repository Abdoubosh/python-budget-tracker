# Python Budget Tracker

Command-line application to track expenses and manage a simple budget using Python and JSON.

## Features
- Add expenses
- View all expenses
- Show total spending
- Delete a specific expense
- Delete all expenses
- Data saved using JSON file
- Input validation

## How to run
1. Install Python  
2. Run the file:
   `python Budget-Tracker.py`

## What I learned
- Working with JSON (save/load data)
- Lists and dictionaries
- Loops and conditionals
- File handling (read/write)
- Error handling (try/except)
